How It Works

Capture
Staff browser tool submits the HTML of the page under review
2
Parse

3
Analyze
CrewAI runs 4 specialist agents in sequence
4
Stream
Explained results return live via Server-Sent Events
Submitted HTML→ /analyze-html/→ text/event-stream→ analyst view
