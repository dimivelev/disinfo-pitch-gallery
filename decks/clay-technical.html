The Problem

Manual disinfo tracing doesn't scale

Analysts face more narrative content than any team can read — in more languages, engineered to slip past keyword filters.

  • Volume. Narratives spread across countless pages far faster than manual review can read them.
  • Evasion. Manipulation is stylistic — loaded headlines, emotional framing, one-sided bias — invisible to keyword and URL blocklists.
  • Multilingual. Threats cross languages; monolingual, region-bound tooling misses them.
  • Surface-level checks. No single signal — source, keyword — captures clickbait, emotion, bias and conspiracy at once.
unstructured · unranked · multilingual · endless

The Core Idea

A crew of specialists, one signal each

Disinfo Analyzer decomposes each page into four independent, measurable signals using a sequential multi-agent crew with a reasoning layer on top.

  • Parse. Newspaper3k extracts headline and body from raw page HTML — analysis runs on content, not page chrome.
  • Decompose. Four CrewAI agents run in sequence, each owning exactly one disinformation signal.
  • Ground. Three agents delegate to fine-tuned transformer classifiers; one applies a structured seven-trait conspiracy framework.
  • Structure. Output is a compact per-signal verdict — clickbait, emotion, bias, conspiracy — not a paragraph of opinion.

The Desktop Workspace

The analyst workflow

An analyst runs a full-page analysis from the browser workspace and watches each signal resolve in real time.

  • One action. The extension captures the active tab's full HTML via chrome.scripting.
  • Off-thread stream. An MV3 offscreen document holds the long-running connection and streams Server-Sent Events; the log updates live.
  • Per-tab state. Each tab's analysis persists in storage.local, surviving panel close and reopen.
  • Readout. A Final Result card lists every agent's verdict, side by side.

The Tech Stack

The inference pipeline

A FastAPI service turns page HTML into four model verdicts, orchestrated by CrewAI and streamed back over SSE.

  • Ingest. POST /analyze-html/ → Newspaper3k parse → a fresh, isolated crew per request.
  • Reasoning layer. Gemini 2.5 Flash (via CrewAI, temp 0.1) directs each agent and writes the conspiracy analysis.
  • Grounded classifiers. Transformers on PyTorch — a multilingual XLM-RoBERTa clickbait model works across languages; ModernBERT for GoEmotions and MBIC political bias.
  • Delivery. Streamed as SSE; shipped as a multi-stage Docker image via GitHub Actions to Google Cloud Run.

05/ 05
Platform & Audience

Built for the desktop, on purpose

Deep disinformation analysis needs screen real estate, secure sessions and sustained attention — not a phone.

  • Deliberate desktop. A Chrome MV3 extension with a streaming panel, built for the researcher's workstation. Mobile is out of scope.
  • Why. Side-by-side reading, long-running streamed analyses and controlled environments suit serious investigative work.
  • Deployment. Containerized FastAPI on Cloud Run, region-pinned — or self-hosted from the same Docker image for data control.
  • Who. Threat-intel & defense teams, research institutes, policy groups, investigative journalists.
