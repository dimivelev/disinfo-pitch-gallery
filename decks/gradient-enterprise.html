How It Works

From a live page to a scored verdict in four steps

1
Capture
MV3 extension (or API call) submits the page HTML
2
Parse
FastAPI extracts headline + body with Newspaper3k
3
Analyze
CrewAI runs 4 specialist agents in sequence
4
Stream
Results flow back live via Server-Sent Events
HTML payload→ POST /analyze-html/→ text/event-stream→ staff UI or pipeline
